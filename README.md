# Insurance Data Pipeline
This project is an end-to-end data pipeline built using Python to process insurance-related datasets. It automates the complete lifecycle of data handling — from ingestion and validation to preprocessing, transformation, and retention.

The pipeline is designed to run daily using scheduling, ensuring continuous and reliable data processing.
